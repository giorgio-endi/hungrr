HUNGRR 
Swipe. Match. Eat.
HUNGRR is a collaborative restaurant decision-making app that helps groups of friends quickly decide where to eat using a swiping system inspired by dating apps. Users can create rooms, invite friends with a code, and swipe through nearby restaurants together. When everyone likes the same restaurant, the app declares a match and suggests the place for the group. The platform also includes a social feature where users can connect with others based on shared food preferences.

Live Demo
https://hungrr-nine.vercel.app

Features
Create and join rooms with a code
Real-time collaborative restaurant swiping
Automatic restaurant matching for the group
User profiles with food preferences
Food-based social matching
Messaging between matched users

Tech Stack
React + Vite (Frontend)
Firebase Authentication (User accounts)
Firebase Firestore (Real-time database)
Google Places API (Restaurant data)
Vercel (Deployment)


Demo Instructions
Open the app on two different browsers or devices.
Create an account on both devices.
On one device, create a room and enter a location.
Share the room code with the other device and join the room.
Start swiping restaurants together.
When everyone likes the same restaurant, the app will declare a match.
